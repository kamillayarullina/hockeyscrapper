# HockeyScrapper Documentation

Welcome to the HockeyScrapper project documentation.

## Project Overview

HockeyScrapper tracks KHL hockey tickets. It scrapes multiple ticket websites, notifies users via Telegram bot when tickets become available, and provides a web dashboard for managing subscriptions and account settings.
